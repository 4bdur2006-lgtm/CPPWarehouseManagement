# C++ Warehouse Management System

## Overview

This project is a **Warehouse Management System** developed in **C++** using object-oriented programming principles. It simulates the core functionality of a warehouse by allowing users to manage products, employees, customer orders and user accounts through an interactive console application.

The system is designed with a modular architecture, separating each component into dedicated classes to improve maintainability, scalability and code organisation. It also supports persistent storage through file handling, allowing data to be saved and loaded between sessions.

---

## Features

The application includes functionality to:

- Manage warehouse inventory by adding, updating, searching and removing products.
- Create and manage customer orders.
- Store and manage employee information.
- Manage user accounts.
- Save and load system data using file handling.
- Navigate the application through an interactive menu-driven interface.

---

## System Architecture

The project follows an object-oriented design, with each class responsible for a specific area of the system:

- **WarehouseSystem** – Coordinates the overall application and menu system.
- **Product** – Represents inventory items and product information.
- **Order** – Handles customer orders and order processing.
- **Employee** – Stores employee details and related operations.
- **User** – Manages user account information.

This modular design improves code readability while making the application easier to extend and maintain.
